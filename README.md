# IRL QR Scanner

Built upon [jsQR](https://github.com/cozmo/jsQR)

Clone this repo and install its dependencies:

## How to use

```bash
git clone https://github.com/IRL2/irl-qr.git
cd nanover-web
npm install
```

Run a live server that opens a new browser tab and refreshes when you edit the code:

```bash
npm run dev
```

Build the standalone web package:

```bash
npm run build
```

Preview the standalone web package:
```bash
npm run preview
```
